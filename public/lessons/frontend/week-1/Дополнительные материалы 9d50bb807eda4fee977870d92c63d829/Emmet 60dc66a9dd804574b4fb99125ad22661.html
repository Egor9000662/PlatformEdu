Emmet

Emmet это встроенный в VS Code плагин, который позволяет быстро набирать код, благодаря использованию большого количества сокращений (аббревиатур). Вы прописываете HTML/CSS код в сокращенном виде и после нажатия определенных комбинаций клавиш, код полностью разворачивается.

Emmet: HTML

Если набрать в редакторе “!” и нажать Tab, то Emmet вставит скелет HTML документа.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8" />
<meta http-equiv="X-UA-Compatible" content="IE=edge" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<title>Document</title>
</head>
<body></body>
</html>

Для подключения CSS файла к HTML документу, достаточно написать link и нажать Tab, чтобы Emmet вставил весь тег.

<link rel="stylesheet" href="">

Чтобы быстро прописать ссылку c http:// - пишем a:link нажимаем Tab.

<a href="http://"></a>

Чтобы добавить тег, нужно просто написать его название и нажать Enter. Например - nav. Если нужен div с классом - пишем точку и название класса. Например .wrapper и жмем Enter:

<div class="wrapper"></div>

Если нужен не div, то пишем нужный тег, точку, название класса - header.top:

<header class="top"></header>

Формула ul>li*5 - развернет ненумерованный список из пяти пунктов, соответственно, если нужен нумерованный, то пишем ol>li*5.

<ol>
<li></li>
<li></li>
<li></li>
<li></li>
<li></li>
</ol>

Чтобы сгенерировать лорем текст на нужное нам количество абзацев, просто пишем p*3>lorem.

<p>
Lorem ipsum, dolor sit amet consectetur adipisicing elit. Tempore sint modi
recusandae quam culpa ea illum?
</p>
<p>
Doloremque adipisci laudantium temporibus distinctio magni veniam deserunt
rerum facilis iure autem repudiandae, similique facere vero excepturi quas
porro.
</p>
<p>
Nesciunt perferendis impedit sint odit doloremque, necessitatibus sequi minus.
Quidem labore sed maxime incidunt, quod obcaecati animi hic voluptas
aspernatur officiis quaerat magnam?
</p>

Emmet: CSS

Аббревиатуры CSS сокращений исходят из названия самих свойств, поэтому здесь все логично.

.top {
mt30
pr10
b
c
}
.top {
margin-top: 30px;
padding-right: 10px;
background: #000;
color: #000;
}
